How to Use

  1. Upload a product image
  2. Set the scale — measure a known reference (e.g. 100 pixels on screen = 10 cm in real life). You can adjust this anytime.
  3. Select dimension type — Length (red, horizontal), Width (blue, horizontal), or Height (green, vertical)
  4. Click and drag on the image to place the dimension line
  5. Adjust text size, re-set scale, or delete individual annotations
  6. Download — perfect for Amazon listings, AliExpress product pages, spec sheets
