How to Use

  1. Upload a product image with text you want to translate
  2. Mark zones — click and drag rectangles over each text block
  3. Type or translate — enter original and translated text, choose font/size/color to match the original style
  4. Preview — the tool erases the original text area and overlays the new translation
  5. Download — export the translated image. All processing is local.
